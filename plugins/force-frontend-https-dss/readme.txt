== Description ==

Allows users to add JavaScript via a WordPress admin page. Ideal for use on WordPress multi-site networks where the site administrators require access to javascript.

<strong>Warning: Do not give users you do not trust access to raw JavaScript. Doing so is a major security risk for your site.</strong>

The creation of this plugin was sponsored by the <a href="http://www.dss.dep.no/">Norwegian Government Administration Services (G.A.S.)</a> and <a href="http://metronet.no/">Metronet</a>.

== Installation ==

Simply install and activate the plugin.

The editing page for adding JavaScript can be found in the "Appearance" menu.

JavaScript will be added to the footer of the page.
